# GC NODE
Backend Cardano node for development purposes

## Get started:

git pull https://github.com/zxpectre/gc_node.git

cd gc_node

./scripts/install.sh

### Preprod

./scripts/up.sh preprod

./scripts/stop.sh preprod

### Mainnet

./scripts/up.sh mainnet

./scripts/stop.sh mainnet
